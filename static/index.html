Chuẩn bị file Excel .xlsx với 3 cột như bên dưới:

Cột A: Số GPLX Cột B: Ngày Sinh Cột C: Loại Bằng

📌 Loại Bằng: ghi PET (nhựa mới) hoặc OLD (giấy cũ). Để trống = PET.
